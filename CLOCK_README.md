See timezone offsets at a glance\n✅ **Persistent Storage** - Settings saved in LocalStorage\n✅ **Responsive Design** - Works on desktop, tablet, and mobile\n✅ **Modern UI** - Retro-style terminal design with neon green theme\n\n## Supported Time Zones\n\n### Asia\n- Tokyo (JST) -
